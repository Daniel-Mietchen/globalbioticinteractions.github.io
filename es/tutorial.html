Comparte tus datos con GitHub y Zenodo

Paso 1

Crea una cuenta personal de GitHub

  1. Visita la pagina de igreso de GitHub.
  2. Ingresa tu nombre de usuario, correo electrónico y contraseña.
  3. Selecciona el tipo de cuenta que requieras, la cuenta gratis funciona bien.
  4. Ahora tienes una cuenta personal de GitHub

Paso 2

Crea una cuenta para tu organización en GitHub

  1. Visita la pagina de igreso de GitHub
  2. Ingresa el nombre de usuario de tu organización, correo electrónico y contraseña.
  3. Selecciona el tipo de cuenta que requieras para tu organización, la cuenta gratis funciona bien.
  4. Transforma la cuenta personal de tu organización en una cuenta para organizaciones de GitHub.
  5. Selecciona el administrador de la organización (pueden ser más de una persona)
  6. Ahora esta cuenta dejó de ser personal y comenzó a ser una cuenta para organizaciones de GitHub

Paso 3

Crea una cuenta de Zenodo

  1. Visita la pagina de igreso de Zenodo.
  2. Crea una nueva cuenta.
  3. Selecciona ingresar con GitHub (Sign up GitHub).
  4. Puedes autorizar ("Grant") a Zenodo que se vincule con tu cuenta de Organizacion de GitHub. Si no lo haces puedes dar la autorizacion desde GitHub.
  5. Autoriza ("Grant") a Zenodo que se vincule con la cuenta de tu organización de GitHub, desde tu cuenta personal/administrador de GitHub.
  6. Tus cuentas están vinculadas.

Paso 4

Crea un repostorio en GitHub

  1. Ingresa a tu cuenta personal de GitHub.
  2. Da click en New.
  3. Nombra y crea un nuevo repositorio.
  4. Ahora tu repositorio está listo par subir tus datos.

  5. Para subir archivos a tu repostiroio es necesaio que des click en "Upload files".

  6. Arrastra o selecciona los archivos que quieras subir a tu repositoro.
  7. Espera a que todos los archivos terminen de cargar antes de abandonar la pagina.
  8. Ahora tus archivos están disponibles en GitHub.

N o t a

En este punto es importate realizar un control de calidad de tus datos. Para más información visita el Tutorial de Travis

Paso 5

Vincula tu repositorio de GitHub con tus cargas de Zenodo

  1. Ingresa a tu cuenta pesonal de Zenodo.
  2. Dentro de Zenodo ve a la pestaña de GitHub
  3. Habilita ("ON") el interruptor para que Zenodo acceda a tu repositorio
  4. Al publicar tu repositorio en Zenodo obtendra un DOI único.
  5. Ahora tu repositorio está publicado en Zenodo

  6. I M P O R T A N T E: El vincular tu repositorio de GitHub con Zenodo facilita el manejo de nuevas versiones de tu base de datos.

  7. Cuando liberes una nueva versión de tu repositorio en GitHub, automaticamente se actualizará la versión de Zenodo.

Paso 6

Avisa a GloBI que tus datos están listos

  1. Ingresa a tu cuenta pesonal de Zenodo.
  2. En tu cuenta ve a cargas (Upload)
  3. Edita tu carga.
  4. Seecciona la comunidad de "Global Biotic Interactions"
  5. Completa todos los metadatos de tu carga.
  6. Publica tu edición.
  7. ¡Felicidades! haz avisado a GloBI que tienes una nueva versión de tus datos.

Si tus datos cumplen los requisitos y pasan el control de calidad de GloBI estarán disponibles en las proximas 24 a 48 horas, de lo contrario nos comunicaremos contigo para poder solucionar el probema y facilitar la carga y formato de tus datos.
