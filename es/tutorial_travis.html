Control de calidad de datos con Travis.org

Tutorial Travis

  1. Asegurate de tener cargado en tu repositorio de GitHub el archivo .travis.yml
  2. Visita la pagina de igreso de Travis.org.
  3. Ingresa (sing in with GitHub) con tu cuenta personal/administrador de GitHub.
  4. Autoriza a Travis ("Grant") que se conecte a tu cueta de tu organización de GitHub.
  5. Activa el repositorio que quieres analizar.
  6. Selecciona el repositorio de interes.
  7. Corre el programa
  8. Si todas tus interacciones tiene el formato correcto obtenras una medalla verde, de lo contrario será amarilla.
  9. Los trabajadores de Travis te informaran en que parte está el problema para que puedas rastrearlo y solucionarlo.
  10. Cuando tus datos tengan la medalla verde están en el formato correcto para que GloBI pueda leerlos ¡Felicidades!
